Markdown 語法

Markdown 是一種輕量級標記語言，通過簡單的標記語法，它可以讓文本內容具有一定的格式。MkDocs 的文件都是由 Markdown 語言編寫的，我們提供了一個基礎教程。

因為這一個關於 MkDocs 的技術文檔， Markdown 語法並不是核心內容，所以這裏隻提供基礎語法的教學，幫助大家快速上手。如果想要學習更多 Markdown 標記語法，可以去 Markdown - 語法說明 學習；如果想要試試功能全麵、操作便捷的 Markdown 編輯器，MdEditor 是個不錯的選擇。建議打開 MdEditor，一邊操作一邊學習。
