介紹： Google App Script (GAS) Google Apps Script 是一個可快速建置開發的後端服務，且可基於Google的服務平台上，簡單、快速地建置商業性服務，他可使用JavaScript語言去做開發建置，也可以透過Browser 直接在網頁是上在Google服務端上開發，亦可以套用Google App，像是Gmail, Calendar, Google Drive...等等，且不用特別安裝，因為這都是Google 所提供的服務。

建立環境： 1- GAS 的 Cheerio 模組

先在 Google Drive 新增一個 GAS 專案，然後點選單上的"資源庫"->"新增程式庫"

2 -建立代碼

function getHappyEvnet()
{
  //爬蟲
  let URL="http://event.happymacao.com/cms62/e/action/ListInfo/?classid=87&page=0";
  options = {muteHttpExceptions: true,'method':'get'};
  let eventAll=new Array();

  let source = UrlFetchApp.fetch(URL, options);
  let html=source.getContentText();
  const $ = Cheerio.load(html,{ decodeEntities: false });
  for (let i=0;i<$('.eventinfo h2').length;i++){
    eventAll.push([$('.eventinfo h2').eq(i).text(),
      $('.eventinfo p').eq(i).text().replace('\n','').replace(/\s+/g,''),
      $('.eventinfo li').eq(i).text(),
      'http://event.happymacao.com'+$('.item a').eq(i).attr('href'),
      'http://event.happymacao.com'+$('.item img').eq(i).attr('src')]);
  }

  console.log(eventAll);    
}
